# Todo Uygulaması

Tarayıcıda çalışan basit bir yapılacaklar listesi. Görevler [Supabase](https://supabase.com) bulut veritabanında saklanır.

## Özellikler
- Görev ekleme, tamamlama ve silme
- Filtreler: Tümü / Aktif / Tamamlanan
- Veriler Supabase'de (Postgres) saklanır

## Çalıştırma
`index.html` dosyasını tarayıcıda aç. Hepsi bu.

## Teknolojiler
- Saf HTML / CSS / JavaScript (derleme adımı yok)
- Supabase (veritabanı + REST API)
